# Jeux de données pour portfolio

## Description

L'objectif de ce repository est de fournir un ensemble de jeux de données pour illustrer des projets de portfolio visant à présenter une activité sous forme de landing page (site vitrine).

## Objectifs du rendu

Dans ce projet, l'objectif est de concevoir et développer une landing concernant l'un des deux jeux de données. Vous choisirez donc de faire une proposition parmi les suivantes : 
- une landing page d'architecte d'intérieur, 
- une landing page de motion designer,
- une landing page de photographe.
